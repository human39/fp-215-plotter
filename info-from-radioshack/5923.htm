FP 215   (26-1193)             Settings                Faxback Doc. # 5923

               SWITCHES  Power Switch
                         OnLine/OffLine Switch
                         Up Arrow Switch      +Y Axis
                         Down Arrow Switch    -Y Axis
                         Right Arrow Switch   +X Axis
                         Left Arrow Switch    -X Axis
                         Pen Up/Down Switch
                         Interface Dip Switch (4 Position)

               SETTINGS  Set Interface Dip Switch
                         Turn Power ON
                         Set Pen Up/Down Switch to Pens Down
                         Set OnLine/OffLine Switch to ONLINE

                         Interface Dip Switch (4 Position) OFF=OPEN
                          1   2   3   4  =  MODE
                         OFF OFF OFF OFF =  Parallel Mode
                         ON  ON  ON  ON  =  600 Baud Serial Mode
                         ON  OFF ON  ON  =  1200 Baud Serial Mode

SWITCH SETTINGS:
SYMBOL  1   2   3   4
Paral   Off Off Off Off
600bps  On  On  On  On
1200bps On  Off On  On
