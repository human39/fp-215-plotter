FP 215   (26-1193)              Features               Faxback Doc. # 5918

The Flat Bed Plotter is an intelligent PLOTTER boasting features such as
graphic function and printer capability and is easy to interface to any TRS
-80 personal computer.

Also, by using a plastic pen recording system, the FP-215 is capable of
crisp and impressive printing and drawing.  By simplifying the mechanism
and incorporating a microprocessor, the plotter has been miniaturized and
offers reliable performance.

Other features include:

*  Figures can easily be drawn using command code and parameter.
*  Self-test and print-functions are provided.
*  A parallel interface and a serial interface are furnished to permit easy
   connection to a computer.
*  High-speed plotting and high graphic quality.
*  Compact, light weight, and space-saving design.
*  Low cost, suitable for a personal computer or a micro computer.

(dkh-08/04/93)
